# ระบบยืม-คืนและติดตามอุปกรณ์ (Equipment Borrow & Tracking System)

ยินดีต้อนรับสู่ระบบจัดการยืม-คืนอุปกรณ์ที่ได้รับการออกแบบใหม่ให้มีความทันสมัยและใช้งานง่าย

## 🌟 คุณสมบัติเด่น (Features)
- **ดีไซน์ระดับพรีเมียม**: แดชบอร์ดใหม่ที่ใช้ดีไซน์แบบ Glassmorphism และไอคอน Lucide ที่สวยงาม
- **รองรับภาษาไทย**: อินเทอร์เฟซทั้งหมดได้รับการแปลเป็นภาษาไทยเพื่อความสะดวกของผู้ใช้
- **ระบบติดตามสถานะ**: ติดตามอุปกรณ์ทั้งหมด, รายการที่กำลังถูกยืม, รายการที่รออนุมัติ และรายการที่เกินกำหนดคืน
- **แถบเมนูข้าง (Sidebar)**: จัดกลุ่มเมนูอย่างเป็นระเบียบพร้อมไอคอนประกอบ
- **ส่วนหัว (Header)**: มีระบบค้นหาและข้อมูลโปรไฟล์ผู้ใช้

## 🛠️ สถาปัตยกรรม (Tech Stack)
- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: .NET Core API, Entity Framework Core
- **ฐานข้อมูล**: PostgreSQL

## 🚀 วิธีการติดตั้ง (Getting Started)

### ส่วนของ Backend
1. เข้าไปที่โฟลเดอร์ `backend/EquipmentBorrow.API`
2. ตรวจสอบไฟล์ `appsettings.json` ว่ามีการเชื่อมต่อฐานข้อมูลที่ถูกต้อง
3. รันคำสั่ง:
   ```bash
   dotnet run
   ```

### ส่วนของ Frontend
1. เข้าไปที่โฟลเดอร์ `frontend`
2. ติดตั้ง dependencies:
   ```bash
   npm install
   ```
3. รันระบบ:
   ```bash
   npm run dev
   ```
   *(หมายเหตุ: หากพบปัญหาเกี่ยวกับเครื่องหมายวรรคตอนในเส้นทางไฟล์บน Windows ให้ใช้ `node "node_modules/next/dist/bin/next" dev -p 3001`)*
